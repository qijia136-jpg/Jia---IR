# C-Level Assistant Tools

这是一组面向 C-Level 高管助理场景的应用工具，覆盖 IR、人事、企业管理和业务细节分析。目标是把高管日常面对的投资人关系、战略招聘、质量管理和经营判断，转化为可交互、可结构化输出、可辅助决策的 Agent 工作台。

它不是单一问答工具，而是一套围绕企业经营核心任务设计的应用场景集合。每个工具都尽量把判断过程拆成可输入、可评分、可排序、可输出的业务模块，帮助管理层更快完成信息整理、优先级判断、方案生成和跨部门沟通。

## 1. IR 功能

IR 模块主要服务于融资、投资人关系管理和战略合作筛选。工具可以根据公司所处行业、融资阶段、地区、技术方向、上市目标和合作偏好，推荐匹配的基金、CVC、药企、MNC、产业平台和长期资本。

核心任务包括：

- 查询具体投资机构的现状，包括投资阶段、关注赛道、交易偏好、地域覆盖、代表案例和公开证据。
- 根据项目画像自动生成投资人推荐，帮助管理层判断哪些机构适合作为融资对象、战略合作方、licensing 或 co-development 伙伴。
- 同时评估财务融资匹配度和战略合作价值，支持 CEO、CFO、BD 和 IR 团队进行融资准备和投资人优先级排序。

对于医疗和 biotech 场景，IR 工具还特别覆盖 ATMP、CAR-T、细胞治疗、基因治疗、递送平台、临床阶段、港股或美股上市路径等业务判断。

对应页面：

- `investor_agent_public_app.html`

## 2. 人事功能

人事模块面向非 HR 背景的高管，重点不是泛泛地“招人”，而是从公司战略反推关键岗位。用户输入公司阶段、战略重点、业务缺口、现有团队基础、HC 数量和到岗时间后，工具会生成招聘优先级、岗位组合、候选人画像、目标公司池、Mapping 方法和面试验证重点。

适用场景包括：

- 战略招聘和高管团队建设。
- 从公司业务阶段反推关键岗位。
- 判断组织当前缺的是科学能力、临床注册能力、质量供应链能力、商业化能力还是资本市场能力。
- 输出人才 Mapping 方法、目标公司分层、目标部门、关键词、Boolean 搜索式、候选人分层和推荐触达路径。

例如，当公司从本土创新药公司走向全球化时，工具可以判断是否应优先补充全球临床开发负责人、注册负责人、质量供应链负责人、商业化负责人、医学事务负责人或 CFO 等角色。

对应页面：

- `index_HR.html`

## 3. 企业管理功能，包括质量管理

企业管理模块覆盖公司运营中的流程、质量、合规和跨部门协同。其中质量管理工具以 CAR-T 医学质量管理为例，构建了一个面向临床、质量、生产和供应链团队的管理工作台。

质量管理 Agent 支持：

- 患者准入评估。
- CAR-T 治疗流程模拟。
- 质量审核。
- 管理报告生成。
- GMP、GSP、内部质量审核、供应商审核、CAPA 复盘等管理输出。

它可以围绕患者基本信息、诊断、既往治疗史和风险因素，输出医学质量提示；也可以模拟从患者评估、细胞采集、生产制备、质量放行、冷链运输、回输监测到随访上报的完整流程。

在管理层场景下，这个工具的价值在于快速识别流程风险、质量闸口、CAPA 路径、跨团队职责和审核发现，为企业运营和质量体系建设提供结构化支持。

对应页面：

- `quality_management_agent.html`

## 4. 其他业务细节

除了 IR、人事和质量管理，这套 C-Level 助理工具还可以扩展到更多业务细节场景，例如：

- BD 合作伙伴筛选。
- 上市地判断。
- 基石投资人识别。
- ODI 执行复杂度判断。
- 战略资本分类。
- 临床阶段匹配。
- 适应症布局。
- 靶点布局。
- 药企外部合作偏好。
- 商业化准备。

整体设计逻辑是：把高管脑子里需要反复判断的问题，拆成可输入、可评分、可排序、可输出的业务模块。每个工具都不只是生成文字，而是尽量把结论变成可执行动作，包括推荐对象、判断依据、风险点、下一步行动、联系人路径、面试问题、审核发现和管理报告。

这套工具可以作为 C-Level 助理的数字化工作台，帮助 CEO、CFO、CHRO、COO、BD 和质量负责人在复杂业务场景中更快形成结构化判断。
